HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title tag is used for SEO mapping for search engines
  • The A with brackets is called an anchor and is used to link with Href
  • ID's and Classes are used to identify certain items for later use in CSS and javascript

CSS

  • CSS stands for cascading style sheets and is the basic style of your web page
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is really annoying
  • Git commands that I know, Git status, Git checkout, -b flags for new branch
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • pwd Print Working Directory
  • CD is change directory
  • mkdir to make a folder
  • touch creates a file
  • mv to move files
  • up and down arrows goes through your command history
  • clear will clear the console
  • list = ls gives a list of files
  • cd .. to go back
  • cd .. && cd .. Brings you to the first folder of the directory
  • *name of file* . will start that file
  • rm will remove file
  • rmdir will remove folder
  • exit will close the CLI
